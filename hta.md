Hierarchical Task Analysis (HTA): Housekeeping 

Robot App Interface 

Goal 0: Direct the housekeeping robot to perform and manage home cleaning tasks through the mobile app for personal home use 

1. Select or define cleaning area 

1.1 Upload a floor plan image of the home (do not list or take picture) 

1.2 Identify and label rooms on the plan (if not done in floorplan) 

1.3 Identify floor / surface spaces in rooms  

1.4 Confirm selected area(s) 

2. Define cleaning task 

2.1 Select area(s) to clean visually using floor plan uploaded 

2.2 Floor Cleaning 

2.2.1 Select floor type (carpet, hardwood, tile, laminate) 

2.2.2 Select cleaning method (vacuum, sweep, mop, polish) 

2.2.3 Choose level of cleaning (light, medium, deep) 

2.2.4 Define any extra actions (clean under furniture, include baseboards, stains, carpet freshener, stain removal) 

2.2.5 Select cleaning product (Swiffer, mop + solution, polish, ...) 

2.2.6 Confirm Selections 

2.3 Surface Cleaning 

2.3.1 Select area type (kitchen, bathroom, living space, bedroom) 

2.3.2 Select cleaning depth (light, medium, deep) 

2.3.4 Choose cleaning method (dust, disinfect, polish, wipe) 

2.3.5 Choose cleaning product (wipes, polish, bleach) 

2.3.6 Choose whether to tidy or organize surfaces (yes/no) 

2.3.7 Confirm Choices 

2.4 Laundry Cleaning


2.4.1 Define Collection: Identify where the laundry needs to be collected from.

2.4.2 Define Sorting:

Sort by color (e.g., separate whites).

Sort by article type or fabric (e.g., delicate vs. heavy).

2.4.3 Configure Wash Cycle:

Select temperature (hot, cold).

Select load size.

Select detergent type (e.g., liquid, pods, dark fabric specific).

Select additives (bleach, fabric softener, scent beads).

2.4.4 Configure Dry Cycle:

Select drying setting (e.g., bulky, delicate).

Set dryer timer (e.g., 5 minutes to freshen up).


2.4.5 Define Post-Wash: Define putting away the laundry.

3. Execution and Monitoring

3.1 Confirm Task:

Review robot feedback stating the task (e.g., "Vacuum the whole house").

Review estimated time for completion.

Confirm intention to proceed.

3.2 Monitor Progress:

View visual progress indicator (progress wheel or bar).

View system processing status.

3.3 Cancel Task:

Select option to cancel or stop the robot.

Receive confirmation that task is canceled.

Return to neutral state/main menu (robot resets).

4. Scheduling and Customization

4.1 Define User Routines:

User defines what "Deep Cleaning" consists of.

User defines what "Light Cleaning" consists of.

User defines what "Medium Cleaning" consists of.

4.2 Schedule Tasks:

Set frequency of tasks (e.g., weekly, daily).

Create a queue or list of tasks to be performed.

5. Additional Requirements (General)


5.1 Floor Plan Interaction: Use the uploaded floor plan image to visually select rooms to clean.


5.2 Cancelation Safety: Ensure user can cancel/back out of operations at any time.
